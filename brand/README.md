# PASSMAP Purple Logo Asset Kit

## Recommended placement in the project
Copy selected files into:

public/brand/

Copy favicon.ico into:

public/favicon.ico

## Main files
- passmap-logo-horizontal-purple.png: primary horizontal logo for white/light UI
- passmap-logo-horizontal-white-wordmark.png: purple symbol + white wordmark for dark sections
- passmap-logo-compact-purple.png: compact header/mobile lockup
- passmap-symbol-purple.png: symbol-only brand icon
- passmap-symbol-purple-192.png / 512.png: PWA/app icons
- favicon.ico: browser tab favicon
- site.webmanifest: optional manifest template
