### 테스트명

- self-report 과대상승 점검형

### 목적

- 경험 입력이 거의 없을 때 강점/일하는 방식만으로 `직무 성격 연결성`이 어디까지 오르는지, 다른 축까지 과도하게 오염시키는지 확인한다.

### 입력 요약

- 목표 직무: 사업기획
- 목표 산업: 컨설팅 / 리서치
- 전공: 사회학
- 자격증: 없음
- 프로젝트: 없음
- 인턴/현장실습: 없음
- 계약직/단기 실무: 없음

### 결과 요약

- 전체 난이도는 렌더 로직 기준 `높음`으로 유지됐다.

### 판정

- PASS

### 핵심 해석

- axis5 과대상승은 존재하지만, 이번 실행에서는 다른 4축 점수까지 오염시키는 수준은 아니었다.
- 현재 더 큰 잔여 리스크는 axis5 점수 오염 자체보다 whyThisRead와 상단 surface에서의 설명 누락이다.
