# PASSMAP Logo Asset Kit

## Included files

### Core
- `passmap-logo-horizontal-color.png` / `@2x`: 기본 가로형 로고. 웹 헤더, 랜딩페이지, 문서 표지, 영상 첫 화면에 사용.
- `passmap-symbol-color.png` / `@2x`: 심볼 단독. favicon, 앱 아이콘, 카드/버튼 아이콘, SNS 프로필에 사용.
- `passmap-wordmark-navy.png` / `@2x`: 텍스트 워드마크 단독. 심볼이 이미 있는 UI에서 보조로 사용.
- `passmap-lockup-compact-color.png` / `@2x`: 작은 공간용 콤팩트 락업.

### Dark background
- `passmap-logo-horizontal-white-wordmark.png` / `@2x`: 어두운 배경용. 심볼은 컬러, 텍스트는 흰색.
- `passmap-wordmark-white.png` / `@2x`: 어두운 배경용 워드마크.

### Mono
- `passmap-logo-horizontal-navy.png` / `@2x`: 단색 네이비 버전.
- `passmap-symbol-navy.png` / `@2x`: 단색 네이비 심볼.
- `passmap-symbol-white.png` / `@2x`: 단색 화이트 심볼.

### Favicon / App icon
- `favicon.ico`
- `passmap-favicon-16.png`, `32`, `48`, `64`, `128`, `180`, `192`, `256`, `512`

## Recommended usage

- Header / landing hero: `passmap-logo-horizontal-color.png`
- Dark hero / video intro: `passmap-logo-horizontal-white-wordmark.png`
- Browser favicon: `favicon.ico` 또는 `passmap-favicon-32.png`
- Mobile home icon / PWA: `passmap-favicon-192.png`, `passmap-favicon-512.png`
- SNS / profile / pitch deck icon: `passmap-symbol-color@2x.png`
- Watermark: `passmap-symbol-white.png` 또는 `passmap-logo-horizontal-white-wordmark.png` with opacity 70~85%

## Important note

This kit was extracted from the supplied raster image and converted into transparent PNG assets. It is suitable for immediate web, deck, and video use. For large outdoor print or strict brand identity work, create a true vector SVG/AI version later from the source design.
