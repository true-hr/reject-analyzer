# 카피/톤 규칙

작성 예정.

## 현재 기본 방향

- `업무 관리`보다 `경험 정리` 톤을 우선한다.
- `최근 리포트`보다 `최근 커리어 인사이트` 톤을 우선한다.
- `이력서 문장`보다 `이력서 후보` 표현을 우선한다.
- 확정형 표현보다 가능성형 표현을 쓴다.
- 사용자가 평가받는 느낌보다, 기록이 커리어 자산으로 정리되는 느낌을 줘야 한다.

## 피해야 할 표현

- 0건
- 없습니다
- 기록 없음
- 무조건 맞습니다
- 이 직무가 맞습니다
- 이력서 문장을 만들었습니다

## 권장 표현

- 경험 신호가 보입니다
- 연결 가능성이 있습니다
- 이력서 후보로 정리해볼 수 있어요
- 기록을 더 남기면 분석 정확도가 높아져요
