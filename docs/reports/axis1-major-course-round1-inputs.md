# Round 1 입력표: ChatGPT 작성 완료본

## 작성 원칙

- 과목은 "실무 경험을 증명하는 근거"가 아니라 "전공을 직무 언어로 번역하기 위한 학습 기반"으로 사용한다.
- 사용자가 실제 수강 과목을 입력하지 않은 경우에는 "어필할 수 있는 과목 예시"로 표현한다.
- 전공 과목만으로 직무 수행 경험이 확인됐다고 단정하지 않는다.
- 각 전공의 개념이 다른 전공/직무에 오염되지 않도록 majorKey 조건을 반드시 둔다.
- Axis1은 전공 중심 축이므로 인턴/프로젝트/공모전 경험은 핵심 근거가 아니라 보조 장면으로만 활용한다.
