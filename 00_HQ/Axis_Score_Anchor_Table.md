왜 이 점수인가

어떤 evidence가 부족하면 여기서 멈추는가
2-2. 점수 앵커가 직접 연결될 대상

2-3. 점수 앵커 문장의 원칙

대분류 B. 축별 1~5점 앵커 표
Axis 1. 전공과 직무의 연결성
보는 질문

전공과 실제 역할 경험이 목표 직무의 핵심 과업과 얼마나 직접 연결되는가

1점
정의

전공과 역할 경험 모두 목표 직무와 직접 연결되지 않음

전형적 evidence 상태
major mismatch
project directness 거의 없음
internship directness 거의 없음
설명 핵심

직무와 이어지는 근거가 전공/경험 양쪽에서 모두 약함

상위 점수 미도달 이유

직무 핵심 과업을 바로 떠올리게 하는 역할 evidence가 없음

2점
정의

전공 또는 경험 중 하나에 약한 연결 단서는 있으나 직접 연결성은 낮음

전형적 evidence 상태
전공은 adjacent/weak
프로젝트나 인턴 중 하나에 부분 연결
role evidence가 모호하거나 약함
설명 핵심

닿는 지점은 있으나 직무 직접성까지는 부족함

상위 점수 미도달 이유

직무와 직접 이어지는 project/intern role evidence가 부족함

3점
정의

전공 또는 프로젝트/인턴 중 하나는 목표 직무와 꽤 맞닿아 있으나, 고점으로 갈 만큼의 role evidence 일관성은 부족함

전형적 evidence 상태
major direct인데 project/intern 약함
또는
major mismatch이지만 project direct 하나 있음
또는
project/intern은 있으나 둘 다 강하진 않음
설명 핵심

핵심 연결 근거는 존재하지만, 직무 직접성을 강하게 뒷받침하는 복수 근거가 부족함

상위 점수 미도달 이유

전공 + 역할 경험 또는 project + internship이 같은 방향으로 정렬되지 않음

4점
정의

전공과 역할 경험 중 둘 이상이 목표 직무와 직접 연결되며, 설명 일관성이 높음

전형적 evidence 상태
major direct
project direct
internship direct 또는 strong support
role evidence가 비교적 명확
설명 핵심

직무와 이어지는 근거가 여러 입력에서 일관되게 확인됨

상위 점수 미도달 이유

아주 강한 실전 role evidence의 밀도나 breadth가 5점급으로 충분하진 않음

5점
정의

전공, 프로젝트, 인턴/실무 근거가 모두 강하게 정렬되어 직무 직접성이 매우 높음

전형적 evidence 상태
major direct
project direct
internship direct
primary evidence source가 한쪽에 치우치지 않고 강하게 정렬
설명 핵심

직무 직접 연결성이 여러 evidence에서 안정적으로 반복 확인됨

상위 점수 미도달 이유

원칙적으로 없음. 다만 coverage/consistency 관점의 약한 ceiling hint만 허용

Axis 2. 산업 분야 이해도
보는 질문

목표 산업의 업무 맥락과 연결되는 evidence가 얼마나 직접적이고 반복적으로 확인되는가

1점
정의

목표 산업과 직접 연결되는 맥락 evidence가 거의 없음

전형적 evidence 상태
major unaligned
certifications unaligned
context none
범용 프로젝트만 있음
설명 핵심

산업 자체를 이해했다고 읽을 수 있는 직접 근거가 부족함

상위 점수 미도달 이유

산업 specific evidence가 없음

2점
정의

산업과 닿는 단서는 있으나, 실제 산업 맥락을 이해했다고 보기엔 근거가 얕음

전형적 evidence 상태
자격증만 있음
전공만 간접 관련
프로젝트 주제만 관련
strong context 부재
설명 핵심

산업 연관 신호는 있으나 단일 경로나 약한 신호 수준임

상위 점수 미도달 이유

산업 맥락을 직접 보여주는 실무/프로젝트/복수 경로 근거가 부족함

3점
정의

목표 산업과 연결되는 유의미한 근거가 1개 이상 있으나, 복수 경로 정렬이나 맥락 깊이는 아직 제한적임

전형적 evidence 상태
major aligned 하나
또는
certification direct 하나
또는
context strong 하나
alignedSourceCount = 1에 가까움
설명 핵심

산업과 이어지는 주된 경로는 있으나, 그 외 경로가 비어 있어 중간 수준으로 읽힘

상위 점수 미도달 이유

복수 경로(전공+실무, 자격+실무 등)의 일관된 산업 맥락이 부족함

4점
정의

산업 관련 근거가 복수 경로에서 확인되고, 목표 산업의 업무 문맥과 꽤 잘 맞닿음

전형적 evidence 상태
major aligned + context strong
또는
certification direct + context strong
또는
strong context + 보조 신호 다수
설명 핵심

산업 맥락 근거가 단일 단서를 넘어 반복적으로 확인됨

상위 점수 미도달 이유

산업 특화성은 충분하지만 coverage나 깊이가 5점 수준으로 매우 강하지는 않음

5점
정의

전공/자격/실무 맥락이 목표 산업과 강하게 정렬되어 산업 이해 근거가 매우 선명함

전형적 evidence 상태
aligned 경로가 2~3개 이상
context strong
산업 specific evidence가 명확
설명 핵심

산업 이해도 근거가 복수 경로에서 강하게 반복 확인됨

상위 점수 미도달 이유

원칙적으로 없음. 약한 ceiling hint만 허용

Axis 3. 실행 깊이와 책임 경험
보는 질문

얼마나 많이 해봤는지가 아니라, 실제로 얼마나 깊이 있게 맡아 수행했고 결과/지속성이 있었는가

1점
정의

실행 깊이와 책임을 보여주는 evidence가 거의 없음

전형적 evidence 상태
outcome none
duration none
evidenceStrength none
comboEvidence 없음
설명 핵심

실제로 맡아 끝까지 수행한 깊이를 읽기 어려움

상위 점수 미도달 이유

결과 수준과 지속 기간 모두 근거 부족

2점
정의

경험은 있으나 참여 수준에 가까워 실행 깊이와 책임 수준은 낮게 읽힘

전형적 evidence 상태
project outcome support
duration short
evidence groups 적음
weak depth
설명 핵심

경험 존재는 확인되지만 깊이 있는 실행까지는 이르지 않음

상위 점수 미도달 이유

결과물 수준 또는 지속 기간이 약함

3점
정의

실행 근거는 있으나 outcome / duration / responsibility 중 하나 이상이 제한되어 중간 수준으로 읽힘

전형적 evidence 상태
project outcome 어느 정도 있음
duration 짧거나 불명확
comboEvidence 없음
evidenceStrength mixed
설명 핵심

실제로 해본 경험은 있으나 깊이와 책임이 고점 수준으로 충분히 누적되진 않음

상위 점수 미도달 이유

결과 수준, 지속 기간, 복합 경험 중 최소 하나가 비어 있음

4점
정의

결과 수준이나 책임 범위가 꽤 강하고, 지속성 또는 복합 경험도 뒷받침됨

전형적 evidence 상태
project outcome strong
duration long 또는 comboEvidence true
evidenceStrength strong/mixed-high
설명 핵심

단순 참여가 아니라 실제로 맡아 수행한 깊이가 비교적 분명함

상위 점수 미도달 이유

결과 수준·지속성·복합 경험이 모두 최고 수준으로 결합된 것은 아님

5점
정의

결과 수준, 지속성, 책임 경험이 모두 강하게 결합되어 실행 깊이가 매우 높음

전형적 evidence 상태
outcome strong
duration long
comboEvidence true
evidenceStrength strong
설명 핵심

실행 깊이와 책임 경험이 여러 속성에서 안정적으로 강하게 확인됨

상위 점수 미도달 이유

원칙적으로 없음. coverage hint만 허용

Axis 4. 타인 상대 경험과 상호작용 적합성

누구와, 어떤 접점에서, 어느 정도 직접적으로 상호작용했는가

정의

이해관계자 접점 evidence가 거의 없음

no_interaction

설명 핵심

협업/고객/사용자/파트너 접점 evidence 없음

2점
정의

협업 경험은 있으나 interaction 강도는 약하고, 주로 내부 또는 간접 접점 수준임

전형적 evidence 상태
support tone
내부 협업 위주
stakeholder 종류 제한적
설명 핵심

타인과의 접점은 있으나 직무상 중요한 직접 interaction으로 읽히기엔 약함

상위 점수 미도달 이유

외부 이해관계자 또는 직접 접점 강도가 약함

3점
정의

의미 있는 interaction evidence는 있으나 외부성/직접성/다양성이 제한적임

전형적 evidence 상태
stakeholder labels 있음
support~부분 strong
external direct contact는 제한적
설명 핵심

상호작용 경험은 확인되지만, 강한 직접 접점이나 다양한 stakeholder 구조까지는 부족함

상위 점수 미도달 이유

외부/직접 stakeholder 경험의 밀도 부족

4점
정의

이해관계자 접점이 비교적 분명하고, 실제 interaction fit이 잘 드러남

전형적 evidence 상태
strong interaction tone
external stakeholder 접점 있음
stakeholder 종류도 어느 정도 확보
설명 핵심

누구와 어떻게 상호작용했는지가 비교적 선명하게 확인됨

상위 점수 미도달 이유

외부 접점의 다양성이나 강도가 5점 수준으로 매우 넓진 않음

5점
정의

다양한 이해관계자와의 직접 interaction evidence가 강하게 반복 확인됨

전형적 evidence 상태
strong tone
external/direct interaction 분명
stakeholder 범위 넓음
설명 핵심

상호작용 적합성이 실제 접점 evidence로 매우 강하게 뒷받침됨

상위 점수 미도달 이유

원칙적으로 없음. breadth hint만 허용

Axis 5. 개인 성향과 강점 적합성
보는 질문

self-report된 strengths/workstyle이 목표 직무 요구와 얼마나 맞고, 실제 경험과 얼마나 정합적인가

1점
정의

강점/workstyle 입력이 거의 없거나, 직무 적합 신호가 매우 약함

전형적 evidence 상태
no_strengths
matchedStrengthLabels 거의 없음
matchedWorkStyleLabels 거의 없음
설명 핵심

직무와 맞닿는 self-report 신호 자체가 부족함

상위 점수 미도달 이유

기본 입력 부족

2점
정의

일부 self-report 신호는 있으나, 매칭이 약하거나 경험 정합성이 거의 없음

전형적 evidence 상태
strength 또는 workstyle 하나만 약하게 매칭
self-report only
experience alignment 없음
설명 핵심

직무 성향과 닿는 단서는 있으나 자기보고 수준을 넘어서기 어려움

상위 점수 미도달 이유

실제 경험과의 정합성 부족

3점
정의

strength/workstyle 매칭은 있으나 전체 aligned signal 수나 경험 정합성이 제한적이라 중간 수준으로 읽힘

전형적 evidence 상태
matchedStrengthLabels 있음
matchedWorkStyleLabels 일부 있음/없음
self-report directly aligned가 약하거나 partial
taxonomy C 또는 A에 가까움
설명 핵심

직무와 맞는 성향 신호는 있으나, 이를 안정적으로 뒷받침하는 정합성이 부족함

상위 점수 미도달 이유

self-report를 넘어서는 일관된 behavior evidence가 부족함

4점
정의

strength/workstyle 매칭이 비교적 강하고 직무 성향과 잘 맞지만, 경험 정합성이나 coverage가 5점 수준까진 아님

전형적 evidence 상태
matchedStrengthLabels 분명
matchedWorkStyleLabels 존재
self-report alignment 양호
다만 type A/B 수준의 제한 일부 존재 가능
설명 핵심

직무 성향과 맞는 self-report 신호가 비교적 분명함

상위 점수 미도달 이유

경험 정합성 또는 workstyle coverage가 완전하진 않음

5점
정의

strength/workstyle 매칭이 매우 강하고 경험 정합성도 충분히 뒷받침되어 self-report 축의 설득력이 높음

전형적 evidence 상태
matchedStrengthLabels 충분
matchedWorkStyleLabels 충분
self-report alignment strong
limiting taxonomy 거의 없음
설명 핵심

직무 성향 적합성이 self-report와 정합성 측면에서 매우 강하게 확인됨

상위 점수 미도달 이유

원칙적으로 없음. 단, self-report 축 특성상 과신 표현은 금지

대분류 C. band별 공통 해석 규칙
2-4. 1~2점 공통 규칙
limiting evidence 중심
positive는 있어도 약한 단서 수준
“기초 근거 부족”을 직접적으로 말해도 됨
2-5. 3점 공통 규칙
가장 중요한 구간
positive 1개 + limiting 1개 필수
“기본 연결은 있으나 고점 근거가 부족”이 핵심
2-6. 4점 공통 규칙
strong positive 1개 필수
“왜 5가 아닌지” limiting 필수
칭찬문 단독 금지
2-7. 5점 공통 규칙
positive 중심
과장/확신 과잉 금지
ceiling hint는 coverage/consistency 정도로만 허용

이 원칙은 현재 조립 계약의 scoreReason pair rule과 그대로 연결됩니다. 문서에서도 3점/4점에서 limiting evidence를 반드시 넣도록 닫혀 있습니다.
